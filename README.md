# Mathweb - 小学自适应数学可视化教学

这是一个面向小学六年级学生的交互式数学教学工具。项目利用动态图形和动画来可视化抽象的数学概念。

## 核心功能

1. **三角形周长**: 三边关系演示
2. **圆的面积**: 割补法推导公式
3. **植树问题**: 间隔与棵数关系
4. **组合图形面积**: 割补与平移
5. **解方程**: 天平平衡演示
6. **列方程解决问题**: 线段图建模
7. **小数乘除法**: 位值原理演示

## 技术栈

- React
- Vite
- Tailwind CSS
- Framer Motion

## 本地运行

1. 初始化环境:
   ```bash
   ./init.sh
   ```

2. 启动开发服务器:
   ```bash
   npm run dev
   ```
